## Overview

Intervals.icu is a free, advanced training analytics platform popular with cyclists and triathletes. FitGlue connects to your Intervals.icu account via API key and can upload your boosted activities. Your enriched workouts — with AI descriptions, heart rate summaries, and performance metrics — appear in your Intervals.icu calendar for detailed analysis.

## Temporarily Unavailable

The Intervals.icu integration is currently **temporarily unavailable**. FitGlue is working on restoring this connection.

## Setup (when available)

1. Go to **Intervals.icu** and log in
2. Navigate to **Settings** → **Developer Settings**
3. Click **Generate API Key** and copy the key
4. Find your **Athlete ID** in the URL: intervals.icu/athlete/**{your-id}**/calendar
5. Enter both values in your **FitGlue Dashboard**

**Note:** The API key and athlete ID are unique to your account. Keep them secure.

## Auth Type: API Key

Intervals.icu uses an API key and Athlete ID — no OAuth required. Simple setup.
